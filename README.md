# 8bit-ALU-Verilog
This project implements an 8-bit synchronous Arithmetic Logic Unit (ALU) in Verilog HDL. The ALU performs arithmetic, logical, and shift operations while generating status flags such as Zero, Carry, and Overflow. The design was verified through simulation and synthesized on a Xilinx XC7Z010CLG400-1 FPGA using Vivado 2025.2.
